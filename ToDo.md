# BlackJack 구성 요소
## 카드
### 일반 카드 
- 2~10
- 스페이스, 클로버, 하트, 다이아몬드 
### 캐릭터 카드
- J, Q, K
- 스페이스, 클로버, 하트, 다이아몬드 
- 10의 값을 가질 수가 있음 

### 에이스 카드 
- A 
- 클로버, 스페이드, 하트, 다이아몬드
- 1 혹은 11의 값을 가질 수가 있음. 

## 참여자
- 21을 넘으면 패
- 상대방보다 숫자가 낮으면 패
- 둘다 버스트면, 딜러가 승 

### 딜러
- 자동으로 카드를 받는 존재
- 17 이상이 될 때까지 카드를 뽑는다.
- 17이상이면 카드 유지
- 상대방과 숫자가 같으면 무승부 

### 플레이어
- 카드 받는 것을 능동적으로 선택할 수 있는 존재 
- 카드 여러 장 추가할 수가 있음
- 상대방과 숫자가 같으면 무승부 
- 버스트할 경우에 자동 패배 

## 컨트롤러 
- 게임 진행 
- 게임 승패 판단

# 게임 규칙
- 모든 플레이어가 카드를 더 받지 않을 때 종료
